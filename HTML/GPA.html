GPA Calculator

Here's how to calculate your GPA:

  1. Enter the mode you want to use. Type Number for percentage grading(i.e 98.76%) or Letter for letter grading(i.e. A-).
  2. Enter the number of classes you have.
  3. Enter your grade as a percentage for each class. Note that there are ten textboxes: you only need to fill the number of boxes that corresponds to the number of classes you have. For example, if you have 1 class you will fill out only 1 box.
  4. Click Calculate to see the result!
